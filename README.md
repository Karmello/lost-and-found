# LAF project

#### Getting project:
`git clone https://Karmello@bitbucket.org/Karmello/laf.git`  
`cd dev/directives/standalone & clone.cmd`

#### Fake data setup:
['js/setup', 'resources/fake-data']

`run app`  
`run setup -d -User` *// empties collection*  
`run setup -c -User` *// fills collection*
